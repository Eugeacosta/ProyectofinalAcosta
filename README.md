# ProyectofinalAcosta
Parte final del proyecto de Python
## Descripción
Este proyecto es una aplicación web desarrollada con Django que permite a los usuarios registrarse, crear blogs, editar perfiles y **enviar mensajes privados entre usuarios**.

## Funcionalidades
- Sistema de Login/Registro.
- Blog: Crear, leer, editar y borrar posts (CRUD).
- Mensajería: Enviar mensajes a otros usuarios y ver bandeja de entrada.
- Perfil: Edición de datos de usuario y avatar.

## Cómo ejecutar el proyecto
1. Clonar el repositorio.
2. Crear entorno virtual y activarlo.
3. Instalar dependencias: `pip install -r requirements.txt`
4. Realizar migraciones: `python manage.py migrate`
5. Crear superusuario (opcional): `python manage.py createsuperuser`
6. Correr servidor: `python manage.py runserver`
